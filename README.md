# Калькулятор опрыскивателя

Простое веб-приложение для телефона и компьютера.

## Что умеет

- считать расход рабочей жидкости по норме вылива
- считать расход на одну форсунку
- подбирать давление по таблице замеров
- работать по участку из справочника или вручную
- хранить участки сада в браузере
- хранить свои замеры форсунки в браузере
- показывать прогноз погоды на 7 дней

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
