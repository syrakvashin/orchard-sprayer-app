# Калькулятор опрыскивателя

Простое веб-приложение для телефона и компьютера.

## Что умеет
- считать нужный объём рабочей жидкости по площади участка
- считать общий расход, расход на 1 форсунку и давление
- хранить свои замеры форсунки
- хранить участки сада

## Локальный запуск
```bash
npm install
npm run dev
```

## Сборка
```bash
npm install
npm run build
```

## Публикация на Vercel
1. Загрузить проект в GitHub
2. Зайти в Vercel
3. Import Project
4. Выбрать этот репозиторий
5. Нажать Deploy

После этого будет публичная ссылка.

## Важно
Данные участков и замеров пока сохраняются только в браузере пользователя через localStorage.
Это значит, что у каждого пользователя будут свои данные.
